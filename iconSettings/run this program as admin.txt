This program installs my Rocketdock Enhanced Settings Utility written in VB6.
If you haven't already done so then you must run this program as administrator
or it may not install properly.

How? Right click on the setup.exe and select run as administrator. That's it.
You can safely ignore this message next time!

At the end of the installation the program should auto-start here but if it
doesn't please understand it is probably Windows 10 and Win10 is a pain in the...
